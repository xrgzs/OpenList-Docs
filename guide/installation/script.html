One-click Script

Requirements:

  • Linux with systemd
  • Root privileges for installation
  • curl, tar installed
  • x86_64 or arm64 architecture

要求：

  • 使用 systemd 的 Linux 系统
  • Root 权限
  • 已安装 curl, tar
  • x86_64 或 arm64 架构
